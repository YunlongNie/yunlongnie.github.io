Chapter 3 fdaWithRMat

Chapter 3 of the book fdawithRMat:

create bspline basis functions

There are normally four relevant arguments that defines a bspline basis system. 1. rangeval: gives the range of the bspline functions 2. nbasis: the number of basis functions in total 3. norder: the order of the bspline, which equals one plus the degree. the default is set to be 4, i.e., cubic splines 4. breaks: the breaking points. It must satisfy that the first and last breaks are the boundary of the rangeval.
